# ZhihuDaily_Demo
A very simple Zhihu Daily App with basic functions. Written in Swift 2.1, Xcode 7.2.1.

![App Demo 1](/Screenshot1.gif)
![App Demo 2](/Screenshot2.gif)


####目前有的功能包括：
- 下拉刷新
- 下拉到底部自动加载更多文章
- 首页标题根据滑动自动更新
- 本地 css 渲染文章
- 记住已读文章并显示灰色

####待添加的功能有：
- 首页图片墙
- 查看文章评论
- 下滑自动加载上一篇文章等
